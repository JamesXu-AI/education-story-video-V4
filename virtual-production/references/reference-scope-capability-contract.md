# Reference and capability contract

Seed Master owns any reference language it chooses in the Prompt and the Shot-local
mapping in the private Segment plan. Virtual production only resolves each unique
literal provider token. Its private namespace is an exact asset-catalog ID,
`roster--member-type`, or the reserved `continuity` namespace.

Every token resolves once, token numbers are contiguous per media kind, and media
counts remain within the refreshed provider profile. Prompt tokens must match the
private-plan token set and be introduced before the first Shot section; the private
plan and runtime media list map those tokens to values.

Resolution never implies semantic compatibility. Compare the resolved catalog
semantics with the private binding responsibility and approved Storyboard. Reject a
reference whose visible or audible facts conflict with the required identity,
appearance/injury, costume, prop, occupancy, environment, group, voice, sound, or
action state even when its namespace and URL are structurally valid.
