# Cinematic Widescreen Production Script: In the Forest

## Production Information

| Field | Value |
| --- | --- |
| Production Type | cinematic_widescreen |
| Genre | Children's educational fable |
| Estimated Runtime Seconds | 157 |
| Target Language | English |
| Target Age Band | older_9_12 |
| Educational Theme | Strength is a trust from Allah and never a right to oppress others. |
| Story Premise | During an Eid forest gathering, a lion claims absolute rule through violence until a mosquito exposes the limit of brute strength and the forest rejects his oppression. |
| Dramatic Strategy | A warm grandfather-and-child frame contains an escalating forest confrontation in which each challenge changes the balance of power through visible action and consequence. |
| Safety and Culture | Islamic belief and Eid context remain source-faithful. Injury and death are clearly readable without gore, exposed tissue, wound fixation, or prolonged suffering. |
| Opening Event | Uthman asks his grandfather to tell him the story of the wicked lion. |
| Ending Event and Obligation | Uthman identifies the lesson, and Grandfather states that strength is a trust that never permits oppression. |
